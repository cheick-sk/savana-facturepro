# Portail Parents

Ce guide vous explique le fonctionnement du portail parents de SchoolFlow et comment accompagner les familles.

## Vue d'ensemble

Le portail parents permet aux familles d'accéder aux informations de leurs enfants : notes, absences, emplois du temps, et de communiquer avec l'école.

## Accès au portail

### Inscription des parents

#### Invitation automatique

Lors de l'inscription d'un élève :
1. Entrez l'email du parent
2. Cochez **"Inviter au portail"**
3. Le parent reçoit un email d'invitation

#### Invitation manuelle

**Parents > Inviter au portail**

1. Sélectionnez le parent
2. Entrez l'email
3. Envoyez l'invitation

### Première connexion

Le parent reçoit un email avec :
- Lien d'activation
- Instructions de connexion

**Étapes :**
1. Clique sur le lien d'activation
2. Crée son mot de passe
3. Accède au portail

### URL du portail

Le portail est accessible à :
- `votre-ecole.saasafrica.com/portal`
- Ou votre domaine personnalisé

## Justifier une absence en ligne

### Procédure pour le parent

1. **Présences > Justifier une absence**
2. Sélectionne l'enfant
3. Remplit le formulaire :
   - Date de l'absence
   - Motif
   - Document justificatif (optionnel)
4. Soumet la demande

### Traitement par l'école

**Présences > Justifications > En attente**

L'administration :
1. Vérifie la demande
2. Valide ou refuse
3. Ajoute un commentaire
4. Le parent reçoit la notification

## Communication

### Envoyer un message

**Parents > Messages > Nouveau message**

Pour contacter :
- Un enseignant spécifique
- La direction
- L'administration

### Réception des messages

Le parent reçoit :
- Messages dans le portail
- Notification par email
- Option SMS (si configuré)

### Notifications automatiques

| Événement | Notification |
|-----------|--------------|
| Nouvelle note | Email immédiat |
| Absence constatée | SMS + Email |
| Bulletin disponible | Email |
| Message de l'école | Email + In-app |
| Rappel de paiement | Email + SMS |

## Application mobile

### Téléchargement

Les parents peuvent télécharger l'app :
- **iOS** : App Store
- **Android** : Google Play

### Fonctionnalités mobiles

- Toutes les fonctionnalités du portail
- Notifications push
- Scan du QR code de l'élève
- Mode hors-ligne partiel

### Configuration

1. Téléchargez l'app SchoolFlow Parents
2. Connectez-vous avec vos identifiants
3. Sélectionnez l'école de vos enfants
4. Activez les notifications

## Administration du portail

### Gérer les accès

**Paramètres > Portail parents > Accès**

| Action | Description |
|--------|-------------|
| **Activer/Désactiver** | Contrôle l'accès d'un parent |
| **Réinitialiser mot de passe** | Envoie un nouveau lien |
| **Modifier les enfants** | Associe/dissocie des élèves |

## Confidentialité et sécurité

### Protection des données

- **Chiffrement** : SSL/TLS
- **Authentification** : Mot de passe + option 2FA
- **Accès restreint** : Uniquement les enfants liés

### Gestion des droits

| Information | Parent | Enseignant |
|-------------|--------|------------|
| Notes de l'enfant | ✅ | ✅ |
| Absences de l'enfant | ✅ | ✅ |
| Emploi du temps | ✅ | ✅ |
| Informations autres élèves | ❌ | ❌ |
| Données médicales | ✅ | Limité |

### RGPD

- Droit d'accès

- Droit à l'effacement
- Portabilité des données

Traitez les demandes de suppression de données.
