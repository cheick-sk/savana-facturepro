# Guide de Démarrage Rapide

## Créer votre compte

### Étape 1 : Inscription

1. Rendez-vous sur [saasafrica.com](https://saasafrica.com)
2. Cliquez sur **"Essayer gratuitement"**
3. Sélectionnez l'application souhaitée
4. Remplissez le formulaire d'inscription :
   - Nom de votre entreprise
   - Email professionnel
   - Mot de passe sécurisé
   - Pays de résidence
5. Validez votre adresse email

### Étape 2 : Configuration initiale

Après validation de votre email, configurez votre organisation :

#### Informations de base
- **Logo** : Téléchargez le logo de votre entreprise (format PNG ou JPG)
- **Nom commercial** : Le nom affiché sur vos documents
- **Coordonnées** : Adresse, téléphone, email

#### Paramètres financiers
- **Devise principale** : XOF, XAF, NGN, GHS, KES...
- **Taux de TVA** : Par défaut 18% (modifiable)
- **Informations fiscales** : Numéro IF/RC (optionnel)

### Étape 3 : Ajouter des utilisateurs

Invitez votre équipe à rejoindre votre organisation :

1. Allez dans **Paramètres > Utilisateurs**
2. Cliquez sur **"Ajouter un utilisateur"**
3. Entrez l'email et définissez le rôle :
   - **Admin** : Accès complet
   - **Manager** : Gestion quotidienne
   - **Utilisateur** : Opérations de base
4. L'utilisateur recevra une invitation par email

## Premiers pas par application

### FacturePro Africa

#### Créer votre première facture

1. Menu **Factures > Nouvelle facture**
2. Sélectionnez ou créez un client
3. Ajoutez des lignes de facturation :
   - Produit/Service
   - Quantité
   - Prix unitaire
   - TVA applicable
4. Vérifiez les totaux
5. Cliquez **"Créer et envoyer"**

#### Personnaliser vos factures

Dans **Paramètres > Modèles de facture** :
- Ajoutez votre logo
- Personnalisez les couleurs
- Configurez les mentions légales
- Activez le pied de page personnalisé

#### Configurer les rappels de paiement

Dans **Paramètres > Rappels** :
- Activez les rappels automatiques
- Définissez les délais (7, 14, 30 jours)
- Personnalisez les messages

---

### SavanaFlow POS

#### Configuration du POS

1. **Créer votre magasin** : Paramètres > Magasins > Nouveau
2. **Ajouter vos produits** :
   - Manuellement via le formulaire
   - Import CSV en masse
   - Scanner de code-barres
3. **Configurer les paiements** : Paramètres > Paiements

#### Première vente

1. Cliquez sur **POS** dans le menu principal
2. Scannez ou recherchez des produits
3. Ajoutez-les au panier
4. Sélectionnez le mode de paiement :
   - Espèces
   - Mobile Money
   - Carte bancaire
5. Imprimez le ticket ou envoyez par SMS

#### Gestion de l'inventaire

- **Alertes stock faible** : Configurez le seuil minimal
- **Réapprovisionnement** : Créez des commandes fournisseurs
- **Inventaire** : Lancez un inventaire physique

---

### SchoolFlow Africa

#### Configuration de l'école

1. **Paramètres > Établissement**
   - Informations de l'école
   - Logo et couleurs
   - Année scolaire active

2. **Créer les classes**
   - Nom (ex: "CM1", "3ème A")
   - Niveau d'étude
   - Capacité maximale

3. **Ajouter les matières**
   - Nom de la matière
   - Coefficient
   - Enseignant assigné

4. **Générer les emplois du temps**
   - Assistant de création
   - Détection automatique des conflits

#### Inscription des élèves

1. **Élèves > Nouvel élève**
2. Remplissez les informations :
   - État civil
   - Parents/Gardiens
   - Classe assignée
3. Générez automatiquement le matricule
4. Imprimez la fiche d'inscription

#### Communication avec les parents

Le **Portail Parents** permet aux familles de :
- Consulter les notes
- Voir les absences
- Payer les frais de scolarité
- Communiquer avec l'école

## Conseils de réussite

### Pour une adoption rapide

1. **Formez votre équipe** : Utilisez les tutoriels intégrés
2. **Importez vos données** : Clients, produits, élèves...
3. **Personnalisez l'interface** : Logo et couleurs de votre marque
4. **Configurez les notifications** : Alertes par email et SMS

### Pour une utilisation optimale

1. **Sauvegardez régulièrement** : Exportez vos données
2. **Surveillez les rapports** : Tableaux de bord hebdomadaires
3. **Formez les nouveaux utilisateurs** : Processus d'intégration
4. **Utilisez l'app mobile** : Disponible sur iOS et Android
